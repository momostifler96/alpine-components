Alpine Components

Bibliothèque headless de composants UI pour Alpine.js 3, stylée avec Tailwind CSS 4. Chaque brique = logique JavaScript + markup de référence à copier.

Alpine.js 3.14+ Tailwind 4 Livewire ready Sans runtime UI

Installation

git clone <url>
cd alpine-components
npm install
npm run dev

Pages disponibles : / (playground), /documentation.html.

Intégration

  1. Copier src/components/ (+ utils/data si InputMask)
  2. Enregistrer les Alpine.data avant Alpine.start()
  3. Importer Tailwind + styles slider
  4. Copier le markup depuis le playground

docs/installation.md — version complète.

Architecture

Modèle headless : la factory retourne état + méthodes ; vous possédez le HTML/Tailwind. Les champs bindables émettent input et change pour Livewire. 

src/components/   → apSelect, apDropdown, …
src/data/         → catalog, phoneCountries
src/utils/        → formatMoney, demoCopy
src/main.js       → bootstrap Alpine

docs/architecture.md

Enregistrement Alpine

import Alpine from 'alpinejs';
import apSelect from './components/Select.js';
import { registerToastStore } from './components/Toast.js';

registerToastStore(Alpine);
Alpine.data('apSelect', apSelect);
// … autres composants

window.Alpine = Alpine;
Alpine.start();

Référence complète : src/main.js
